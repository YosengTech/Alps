关于我们

卓越品质，值得信赖

　　我公司位于福建省省会城市福州东郊，美丽的滨海城市长乐市地理中心鹤上镇，公司成立于1988年，至今有25年的历史。经过多年的潜心经营，公司发展蒸蒸日上，规模不断扩大。现拥有自有土地面积12万平方米，厂房面积5万平方米。现有员工300多人，中高级技术人员70人。 　　公司主要产品为热轧工字钢、槽钢、角钢以及经热镀锌板制成的各种规格圆管、方管、矩管。公司积极奉行“质量第一、用户至上”宗旨，产品质量严格按照Q/CLYS006标准控制，使我公司产品始终保持在行业的前列，产品远销全国各地和出口至菲律宾、印尼、泰国等东南亚一带，深受广大用户好评。在此，竭诚欢迎海内外新老客户与公司真诚合作，共同开拓美好的未来。
